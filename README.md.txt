 Project Overview

This project analyzes student academic data and builds machine learning models to predict whether a student will pass or fail based on academic and behavioral factors.

The goal is to demonstrate data analysis, visualization, preprocessing, and basic machine learning skills.

 Objective

* Analyze student performance data
* Identify key factors affecting academic results
* Build a machine learning model to predict Pass / Fail

 Dataset

* Student academic dataset containing:

  * Gender
  * Study time
  * Absences
  * Past failures
  * Family & school support
  * Internal marks (G1, G2)
  * Final marks (G3)

* Target variable:

  * 'pass_fail'

 Exploratory Data Analysis (EDA)

* Study time vs performance
* Absences vs performance
* Gender vs performance
* Internal marks vs final marks

 Key Insights:

* Higher study time generally leads to better performance
* More absences increase failure chances
* Internal marks strongly influence final results
* Gender has minimal impact compared to academic factors

 Machine Learning Models Used

* Logistic Regression (Primary model)
* Decision Tree (Comparison model)

Model Comparison:

* Logistic Regression showed better and more stable accuracy
* Decision Tree was interpretable but slightly less accurate


 Results

* The Logistic Regression model achieved good accuracy in predicting student performance
* The model can help identify students at risk of failing early
